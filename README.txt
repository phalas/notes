SK
Závislosti
- jQuery v1.11.1 (https://jquery.com/)
- Annotator v1.2.10 (http://annotatorjs.org/)
- PrintArea v2.4.0 (https://plugins.jquery.com/PrintArea/)
- Moodle v2015051100 (https://moodle.org/) (embedded jQuery)

Nastavenia
- Firefox tlač obrázkov na pozadí (https://support.mozilla.org/sk/kb/how-print-web-pages-firefox)
- IE tlač obrázkov na pozadí (Nástroje -> Tlač -> Nastavenie strany -> Tlačiť obrázky na pozadí)

Podporované prehliadače
- Chromium (Version 48.0.2564.82 Ubuntu 14.04 (64-bit))
- Chrome (Version 48.0.2564.103 m Windows 7 (64-bit))
- Firefox (Version 44.0 Ubuntu 14.04 (64-bit))
- Firefox (Version 44.0.1 Windows 7 (64-bit))
- Opera (Version 35.0 Ubuntu 14.04 (64-bit))
- Opera (Version 35.0 Windows 7 (64-bit))
- Internet Explorer (Version 11 Windows 7 (64-bit))
- Safari (Version 5.1.7 Windows 7 (64-bit))

Inštalácia
- Štandardná inštalácia modulov (Site administration -> Plugins -> Install plugins)

Známe obmedzenia
- Opera farby na pozadí sú šedé (Pri tlači do PDF súboru je farba žltá)
- Užívateľ musí mať zapnutý javascript
- Shift+Enter zariadkovanie pri tvorbe poznámok, Enter uloženie poznámky

EN
Dependencies
- jQuery v1.11.1 (https://jquery.com/)
- Annotator v1.2.10 (http://annotatorjs.org/)
- PrintArea v2.4.0 (https://plugins.jquery.com/PrintArea/)
- Moodle v2015051100 (https://moodle.org/) (embedded jQuery)

Settings
- Firefox Print Background Colors (https://support.mozilla.org/en-US/kb/how-print-web-pages-firefox)
- IE Print Background Colors (Tools -> Print -> Page setup -> Print Background Colors and Images)

Supported Browsers
- Chromium (Version 48.0.2564.82 Ubuntu 14.04 (64-bit))
- Chrome (Version 48.0.2564.103 m Windows 7 (64-bit))
- Firefox (Version 44.0 Ubuntu 14.04 (64-bit))
- Firefox (Version 44.0.1 Windows 7 (64-bit))
- Opera (Version 35.0 Ubuntu 14.04 (64-bit))
- Opera (Version 35.0 Windows 7 (64-bit))
- Internet Explorer (Version 11 Windows 7 (64-bit))
- Safari (Version 5.1.7 Windows 7 (64-bit))

Installation
- Standard moodle plugin installation (Site administration -> Plugins -> Install plugins)

Known Limitations
- Opera print background color is gray (Print to PDF background color is yellow)
- User must have javascript turned on
- Hotkey Shift+Enter for new line, Enter for save comment
